Creating More Users

Once registered with Gonebusy your own User account has been created for you. You can now make changes via API to your own User account as well as create/update other User accounts.

For any User accounts that you create via API, you are the Account Manager for those User accounts.

Likewise, any User account created via API could also obtain their own API Key and manage their own account as well as any additional User accounts that they create via the API.

Create a User

Here is a sample POST request to create a new User:

Remember: Substitute sandbox.gonebusy.com below if you are using the Sandbox.

Request:

{% highlight ruby %} https://beta.gonebusy.com/api/v1/users/new.json {% endhighlight %}

with POST body:

{% highlight ruby %} "email" : "tutor@youremail.com", "first_name" : "Private", "last_name": "Tutor", "business_name": "Tutors-R-Us", "external_url": "tutors-r-us.sample.biz", "timezone": "EST" {% endhighlight %}

Response:

Here is the response to the request, note the ID of our new User:

{% highlight ruby %} "Status": "201 Created" { "user": { "id": 54, "email": "tutor@youremail.com", "first_name": "Private", "last_name": "Tutor", "disabled": false, "business_name": "Tutors-R-Us", "external_url": "tutors-r-us.sample.biz", "permalink": null, "timezone": "EST", "created_at": "2016-10-04T19:46:26Z", "updated_at": "2016-10-04T19:46:26Z", "resource_id": 267, "account_manager_id": 22, "address": null } } {% endhighlight %}
