Get Started With Gonebusy

1

Obtain an API Key

Sign Up and become a Pro User to obtain an API Key.

API Keys look like this:

{% highlight ruby %} ac98ed08b5b0a9e7c43a233aeba841ce {% endhighlight %}
2

Configure Your Client

Here we use the Ruby SDK to configure our API Key:

{% highlight ruby %} Gonebusy::Configuration.api_key = ac98ed08b5b0a9e7c43a233aeba841ce {% endhighlight %}
3

Create a Service or Activity

Create the Service or Activity that you would like your clients to book:

{% highlight ruby %} service = Gonebusy::Service.create( name: 'Executive Car Wash', duration: 30, description: 'Get your car washed by executives.' ) {% endhighlight %}
4

Create a Schedule

Create a Schedule for your new Service:

{% highlight ruby %} Gonebusy::Schedule.create( service, start_date: '2016-02-01', end_date: '2016-03-31', start_time: '12pm', end_time: '2pm', days: ['monday', 'wednesday', 'friday'], recurs_by: 'weekly' ) {% endhighlight %}
5

Check Your Work!

Let's check our work by Searching for available time for our Service:

{% highlight ruby %} slots_result = Gonebusy::Service.available_slots( service, date: '2016-03-11' ) puts slots_result > { date: '2016-03-11', slots: [ '2016-03-11T12:00:00Z', '2016-03-11T12:15:00Z', '2016-03-11T12:30:00Z', '2016-03-11T12:45:00Z', '2016-03-11T13:00:00Z', '2016-03-11T13:15:00Z', '2016-03-11T13:30:00Z'] } {% endhighlight %}
